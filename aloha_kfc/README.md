# aloha_kfc

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```



完成的功能：

- [x] 商品首页
- [x] 详情页
- [x] 购物车
- [x] 搜索
- [x] 订单管理
- [x] 地址管理
- [x] 登录注册



技术栈：Vue + Vuex + Axios + Express + 数据库json本地存储
